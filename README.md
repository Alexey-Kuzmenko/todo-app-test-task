# Acid Sof test task

## Todo app features:

1. Add new tasks through an input field and "Add" button.
2. Display a list of created tasks.
3. Delete individual tasks.
4. Mark tasks as completed or uncompleted.
5. Clear the entire task list.
6. Persist tasks between app restarts.
